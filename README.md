# Validação de Formulário (Contato)

Aplicação web desenvolvida com foco em validação de dados no front-end, utilizando JavaScript.

## 🚀 Funcionalidades
- Validação de campos obrigatórios
- Verificação de formato de e-mail
- Validação de tamanho mínimo de nome
- Feedback visual para o usuário

## 🛠 Tecnologias
- HTML
- JavaScript
